# Tetromino Puzzle

A browser-based puzzle game where you fill a grid using all 7 standard tetrominoes. Every puzzle is guaranteed solvable — the board is generated by a backtracking solver before you start.

**[Play it here](https://quasarbright.github.io/tetromino-game/)**

## How to play

- **Select a piece** from the tray on the right
- **Hover** over the grid to preview placement
- **Click** to place the piece
- **Z / X** (or the buttons) to rotate CCW / CW
- **Click a placed piece** to pick it up and move it
- **Esc** or click outside the grid to deselect
- Fill the entire grid to win

## Difficulty

| Mode   | Grid  | Pieces |
|--------|-------|--------|
| Easy   | 4×4   | 4      |
| Medium | 6×6   | 9      |
| Hard   | 8×8   | 16     |

## How it works

Each puzzle is generated by a backtracking solver with two pruning heuristics:

- **Pocket size** — any isolated empty region whose cell count isn't divisible by 4 is unsolvable; prune immediately
- **Checkerboard parity** — on a black/white checkerboard coloring, each region's imbalance must be achievable given the available piece types

The solver finds an 8×8 solution in ~1ms. Piece type variety is encouraged by sorting candidates toward less-used types at each step.
